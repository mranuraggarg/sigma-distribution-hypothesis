# Sigma-Distribution Hypothesis

## Overview
A probabilistic framework for understanding lifestyle disease onset, aging, and prevention using sigma-based positioning within population distributions.

## Quick Start

### R-based validation (recommended)
```bash
# from repo root
Rscript analysis/r/01_sigma_validation.R
```
This generates all illustrative outputs under `analysis/outputs/`.

### Python (figure-only, optional)
```bash
python code/public_database_illustration.py
```

## Reproducibility

### Analysis outputs
All analysis results in the paper and supplementary material are generated by:
```bash
Rscript analysis/r/01_sigma_validation.R
```

Outputs are written to `analysis/outputs/` and include:
- CSV summaries (human-readable)
- PNG figures (GitHub-renderable)
- Text logs documenting the run

Binary `.rds` files are intentionally excluded from version control.

## Note on Data and Scope

This repository does not include restricted or proprietary medical microdata.
Analyses use:
- Synthetic data (internal coherence checks)
- Public NHANES demonstration variables available via the NHANES R package

All results are illustrative and support a methodological hypothesis.
No clinical prediction or medical claims are made.
